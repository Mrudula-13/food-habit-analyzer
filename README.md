# Smart Food Habit Analyzer

A Machine Learning based web application that predicts whether a user's food habits are Healthy, Average, or Unhealthy.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning
- Scikit-learn

## Features
- Login Page
- ML Prediction
- Interactive UI
- Food Habit Classification
- Charts & Visualization

## ML Algorithm
Decision Tree Classifier
